# nBot Help

nBot is a Google Chat bot for friend group chats.

## Commands

**ping**
Check if nBot is alive.
> ping → pong

**roll NdN**
Roll dice. Specify number of dice and sides.
> roll 2d6 → 🎲 [4, 3] = 7

**frink / frinkiac \<quote\>**
Search for a Simpsons scene matching a quote and return a captioned screenshot.
> frink d'oh

**morb / morbotron \<quote\>**
Search for a Futurama scene matching a quote and return a captioned screenshot.
> morb good news everyone

**\<twitter/x link\>**
Paste an X/Twitter link and nBot will return a preview card of the tweet.
