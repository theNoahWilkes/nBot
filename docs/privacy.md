# Privacy Policy

**nBot** is a Google Chat bot for personal use among friends.

## Data Collection

nBot does not collect, store, or share any personal data. 

When you send a message to nBot:
- The message text is processed in memory to determine if a plugin should respond
- No message content, user identifiers, or metadata are written to disk or transmitted to third parties beyond what is necessary to fulfill the request (e.g. sending a search query to Frinkiac or Morbotron)
- No data is retained after the response is sent

## Third Party Services

Some plugins interact with third-party services:

- **Frinkiac** (frinkiac.com) — Simpsons scene search. Search queries are sent to their public API.
- **Morbotron** (morbotron.com) — Futurama scene search. Search queries are sent to their public API.
- **fxtwitter** (api.fxtwitter.com) — Tweet metadata lookup. Tweet URLs are sent to their public API.

These services have their own privacy policies.
